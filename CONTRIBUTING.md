# Contributing to Setrsoft

Thank you for your interest in Setrsoft! This project is a collaborative ecosystem and we need all types of skills (developers, 3D designers, route-setters, climbers).

By contributing, you agree that your contributions will be distributed under the project's **AGPL-3.0** license.



## How can you help?

### 1. Development (This repo)
The project uses:
- **Backend:** Django (with PostgreSQL)
- **Frontend:** React + Vite + Tailwind CSS
- **Infrastructure:** Docker & Nginx

### 2. [Holds database](https://setrsoft.github.io/holds-dataset-hub/)
We are building a global database of climbing holds. You can:
- Share 3D scans of real holds (.obj, .stl, .glb).
- Help clean up meshes (Blender) to optimize editor performance.
- Identify brands and models of existing holds in the Hub.

### 3. Documentation & Translation
- Improve this guide or the technical documentation.
- Translate the interface into other languages to make the tool universal.
- Help us manage the community and global project organisation and development.



## Development Process

### Local Setup
1. Fork the repository.
2. Clone your fork: `git clone <FORK URL>`
3. Launch the environment with Docker: `docker-compose up`

### Git Workflow
- Create a branch for your feature: `git checkout -b feature/my-feature-name`
- Make clear and descriptive commits.
- Push your branch and open a **Pull Request (PR)**.



## 📏 Quality Standards

### Code
- **Tests:** If you add a critical feature, please try to include a unit test.
- **Use of AI:** Add a .md file related to your implementation to allow ai tools to give better results for further implementation.



## 🤝 Code of Conduct
We are committed to maintaining an inclusive and respectful environment. Please be kind during code reviews and Issue discussions.

## 💬 Any questions?
Feel free to open an **Issue** to discuss a new feature or report a bug, or join our community at [![Discord](https://img.shields.io/badge/Discord-Join-7289da?logo=discord&logoColor=white)](https://discord.gg/BdyfNU9TpR).
